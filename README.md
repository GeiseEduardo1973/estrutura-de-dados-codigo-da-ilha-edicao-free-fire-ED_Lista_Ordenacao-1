📖 O "Read" das 3 Versões: 

😒A Evolução do CódigoVersão 1: O Recruta (Lógica Básica)
Nesta fase Novato, o foco costuma ser apenas na entrada e saída de dados.
O que fazia: Cadastro simples de itens (nome e quantidade).
Estrutura: Variáveis isoladas ou arrays simples.
Objetivo: Praticar comandos de repetição e leitura de dados.
Versão 2: O Aventureiro (Organização e Estrutura)

😁Aqui o código começa a ganhar "corpo" de software real.
O que fazia: Armazenava múltiplos itens de forma organizada.
Estrutura: Uso de structs para agrupar informações (ex: nome e tipo) e funções básicas para exibir os dados.
Objetivo: Aprender modularização (dividir o código em funções) e manipulação de conjuntos de dados.

😎 Versão 3: O Mestre (Algoritmos e Performance)
Esta é a versão definitiva que estamos finalizando agora. 
É o momento decisivo onde o jogador precisa de um plano estratégico para escapar da ilha.
O que faz: Montagem de uma torre de resgate com componentes que possuem nome, tipo e prioridade.
Diferencial Técnico: Implementa três algoritmos de ordenação diferentes (Bubble, Insertion e Selection Sort) para organizar os componentes.
Busca Inteligente: Introduz a Busca Binária, que é muito mais rápida, mas exige que o sistema esteja ordenado por nome para funcionar.
Análise Científica: O sistema agora conta o número de comparações e mede o tempo de execução exato (usando clock()), permitindo analisar qual estratégia é mais eficiente.
